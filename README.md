Plateforme Artisanale

Plateforme Artisanale est une marketplace académique développée avec Symfony 6.4, PHP, HTML, CSS, JavaScript et Doctrine ORM.

Cette plateforme permet aux artisans de vendre leurs produits faits main et aux utilisateurs de parcourir, ajouter au panier et commander ces produits.

Fonctionnalités principales:

-Authentification sécurisée avec gestion des rôles : Admin, Artisan, Utilisateur

-Catalogue de produits artisanaux avec gestion du stock

-Gestion du panier et des commandes pour les utilisateurs

-Paiement en ligne via Stripe

-Remboursements automatiques si le paiement dépasse une date limite définie

-Génération de PDF pour les informations utilisateur et commandes

Interface intuitive et responsive pour tous les appareils

Ce projet a été développé dans le cadre d’un projet académique pour fournir aux petits artisans une opportunité de vendre leurs créations en ligne facilement et de gérer leurs ventes en toute sécurité.
